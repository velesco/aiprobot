# `aipro tui`

Open the terminal UI connected to the Gateway.

Related:
- TUI guide: [TUI](/tui)

## Examples

```bash
aipro tui
aipro tui --url ws://127.0.0.1:18789 --token <token>
aipro tui --session main --deliver
```
